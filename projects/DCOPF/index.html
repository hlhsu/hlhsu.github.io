Abstract

The DC Optimal Power Flow (DC OPF) problem is one of the most important optimization problems in power system operations, and the objective of DC OPF is to minimize the total generation cost with respect to power flow and generation constraints. Power system operators solve the DC OPF periodically, e.g., every five minutes, since the load profile and generation cost coefficients (generation cost is equal to generation cost coefficient multiplied by the amount of generated power) can change in real-time. However, obtaining DC OPF solutions is challenging owing to a number of constraints imposed on them, particularly the line flow constraints, which restrict the maximum allowable power flow on each transmission line. Researchers have observed that only a fraction of the line flow constraints ever become active at the optimal solution, despite variations in the load and cost coefficient profiles. This means that only a portion of lines have power flows at their limits. This observation implies that many line flow constraints are redundant/never binding, which can be removed to reduce the complexity of the DC OPF problem. We have developed transmission line constraint-screening algorithms using four different machine learning techniques. We selected neural network as the most suitable method owing to the optimal trade-off between FNR and accuracy it provides on the test set for both 39-bus and 118-bus systems.

Abstract

Working memory or short memory, the ability to hold information in mind during the ongoing flow of inputs, is one of the most important consecutive functions that enable human beings to achieve every type of daily life behavior. Working memory quality and capacity directly correlate to other cognitive behaviors such as attention, motor performance, long-term memory encoding and etc.. investigating the neural foundation behind it is not only important for understanding the core question of cognitive neuroscience -- how does the brain trigger behaviors but also beneficial for providing insights to understanding intelligence differences and even treatment for cognitively deficit diseases. The literature on brain and working memory has already identified brain areas such as the prefrontal cortex and basal ganglia including the thalamus and caudate that are involved in representing and holding information during a working memory task. In this study, we tested functional connectivity among human brains during both working memory task and rest states and how that is associated with individual differences in working memory performance. We identified that poor performance during working memory was represented by a flat and loosely connected brain network whereas during good performance in a working memory task, the brain, especially the cortex and thalamus was strongly connected. We found no difference between groups in the resting state network.
